# Budget Tracker — Personal & Business Edition

A clean, easy-to-use Excel budget tracker for managing
monthly income, expenses, and savings goals.

## Features

- Monthly income and expense logging
- Category-wise spending breakdown (Food, Rent, Transport, etc.)
- Savings goal tracker with progress bar
- Over-budget alerts with conditional formatting
- Monthly summary dashboard with charts
- Yearly overview sheet

## Tools used

- Microsoft Excel
- Formulas (SUMIF, IF, VLOOKUP)
- Conditional Formatting
- Charts & Sparklines
- Data Validation (dropdowns)

## How to use

1. Download the Excel file
2. Enter your monthly income in the Income sheet
3. Log daily expenses in the Expenses sheet
4. View your summary on the Dashboard tab
